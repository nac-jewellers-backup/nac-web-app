#PropTypes Required
    1)Settings - is the props where you define the properties of the carousel

        ##Example

                const dataCarousel = {
                dots: false,
                infinite: true,
                autoplay: true,
                speed: 1000,
                fade: true,
                arrows: false
                }

    2)fadeImages - is the props where you will send the data's such as Video url, image url.
          
         ##Example
                const fadeImages = [
                'https://assets-cdn.stylori.com/120x120/images/product/SR0986/SR0986-1Y.jpg',
                'https://assets-cdn.stylori.com/120x120/images/product/SE0464/SE0464-1Y.jpg',
                'https://assets-cdn.stylori.com/images/megamenu/ear-jacket.jpg'
                ];
