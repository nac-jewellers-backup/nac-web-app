# Carousel
## List of PropTypes
1. **Settings** - *Type Object*
    <br>    
    Configure the settings of the carousel
    >### Example
    >
    >  ```jsx
    >   const dataCarousel = {
    >       dots: false,
    >       infinite: true,
    >       autoplay: true,
    >       speed: 1000,
    >       fade: true,
    >       arrows: false
    >   }
    > ```
    >
2. **fadeImages** - *Type Array of Objects*
    <br>
    to send the data's such as Video url, image url
    >### Example
    >
    >  ```jsx
    >   const fadeImages = [
    >            'https://assets-cdn.stylori.com/120x120/images/product/SR0986/SR0986-1Y.jpg',
    >            'https://assets-cdn.stylori.com/120x120/images/product/SE0464/SE0464-1Y.jpg',
    >            'https://assets-cdn.stylori.com/images/megamenu/ear-jacket.jpg'
    >            ];
    > ```
    >
